How To Properly Select Enclosure For Disk Replacement

Open Storage Array Profile in Autosupports and select Drives section

  1. What is the Current Data Rate listed?.

    2. If 6, go to step 2.

    If 12, go to step 5.

  2. Is there a Drawer column?

    3. If yes, DE6600.

    If no, go to step 3.

  3. Are there more than 12 drives listed in the same tray?

    4. Yes -> DE5600.

    No -> Step 4

  4. If you search the Product ID, is the DE1600 listed?

    5. Yes -> DE1600.

    No -> Go back to step 1 and try again.

  5. If you search the Product ID, is the DE212C listed?

    6. Yes -> DE212C.

    No -> Go back to step 1 and try again.
